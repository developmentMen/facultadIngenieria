	|Clasificacion de la materia|

Materia:
	Sustancia
	-composicion fija
	-no puede separarse por metodos fisicos
	-temperatura constante durante el cambio de estado
		Elementos
			-sustancias simples
			-no pueden descomponerse
		Compuestos
			-formados por dos o mas elementos
			-pueden descomponerse

	Mezclas
	-composicion variable
	-pueden separarse por metodos fisicos
	-temperatura variable durante el cambio de estado
		Soluciones (mezclas homogeneas)
			-una fase
			-formadas por dos o mas sustancias
		Mezclas heterogeneas
			-dos o mas fases
			-formadas por una, dos o mas sustancias

Sustancias
	una sustancia es una porcion de materia constituida por identicas
	(o muy parecidas) unidades elementales. estas unidades elemntales
	pueden ser atomos, iones o moleculas

	Simples
		formada por particulas del mismo elemento

		Alatropia
			una sustancia simple se puede representar
			en diferentes variedades alotropicas
			esta formado por elementos iguales pero
			ordenados espacialmente de una manera distinta
	Compuestas
		formadas por particulas de distintos elementos
	Sustancia Pura
		sistemas homogeneos de una sola sustancia
	Soluciones
		sistemas homogeneos de mas que una sustancia
Mezclas
	Sustancia pura/Homogeneidad
		formados por mas de una sustancia
	Heterogeneidad
		formados por un solo componente y por dos
		fracciones de materia con propiedades diferentes
